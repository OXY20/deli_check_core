# 文档目录

## 版本发布记录

### v1.0.0（2020-04-14）

#### 核心功能
- 支持 `.xls` / `.xlsx` 双格式考勤文件解析
- 自动识别员工信息块，动态提取工号、姓名、部门
- 跨行数据自动合并解析
- 单文件 / 目录批量扫描两种模式
- 输出结构化 JSON（`records.json`、`summary.json`）

#### Web 界面（`deli_check_web` 分支）
- 浏览器端上传考勤文件
- 按人员查看打卡记录详情
- 用餐统计功能：按时段统计早餐 / 午餐 / 晚餐次数
- 导出用餐数据为 `.xlsx` 表格
- 支持开发模式热重载（`DELI_WEB_DEV=true`）

#### 分支说明
- `main`：CLI 核心解析工具主分支
- `deli_core`：核心解析分支（与 `main` 同步）
- `deli_check_web`：Web 界面主基线分支
- `develop_deli_check_web`：Web 界面日常开发分支

此目录用于存放项目相关文档和说明。
